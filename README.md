# Веб-сервис «Заявки в ремонтную службу»

Тестовое задание. Приложение для приёма и обработки заявок, реализованное на **Laravel 11**, **PostgreSQL** и **Docker**.

Проект запускается одной командой. Убедитесь, что у вас запущен Docker Desktop.
