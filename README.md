# Веб-сервис «Заявки в ремонтную службу»

Тестовое задание по реализации системы управления заявками с защитой от "состояний гонки" (Race Condition).

## 🚀 Быстрый запуск (Docker Compose)

Проект настроен для запуска одной командой. Внутри контейнера автоматически выполняются: установка зависимостей, генерация ключей, миграции и наполнение базы (seed).

```bash
docker compose up --build
